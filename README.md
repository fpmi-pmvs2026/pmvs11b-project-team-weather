# Project Name: Weather & Trip Planner (pmvs-weather-trip-planner)

## Description
Мобильное приложение для ОС Android, предназначенное для агрегации данных о погоде и планирования путешествий. Приложение позволяет пользователям искать актуальный прогноз погоды для различных городов, сохранять карточки будущих поездок в локальную базу данных, а также получать push-уведомления с предупреждениями о неблагоприятных погодных условиях.
