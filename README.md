# Project Name: Weather & Trip Planner (pmvs-weather-trip-planner)

## Description
Мобильное приложение для ОС Android, предназначенное для агрегации данных о погоде и планирования путешествий. Приложение позволяет пользователям искать актуальный прогноз погоды для различных городов, сохранять карточки будущих поездок в локальную базу данных, а также получать push-уведомления с предупреждениями о неблагоприятных погодных условиях. 
  
### Основные фичи (что будет уметь приложение):
* **Актуальная погода онлайн:** Вбиваешь любой город — приложение мгновенно связывается с сервером погоды (через OpenWeatherMap API) и показывает текущую температуру, ветер и облачность.
* **Календарь поездок:** Возможность добавлять свои будущие выезды (например: *«Выезд на шашлыки в Заславль 25 мая»* или *«Поездка к бабушке в Брест»*). Все планы сохраняются в базу данных на телефоне и доступны даже без интернета.
* **Удобные жесты:** Список поездок интерактивный — если планы изменились, карточку путешествия можно удалить одним плавным свайпом влево (*Swipe-to-dismiss*).
* **Умные уведомления (Киллер-фича):** Приложение в фоновом режиме проверяет метеоусловия для мест ваших запланированных поездок. Если синоптики внезапно передают шторм, ливень или сильный снегопад в городе, куда вы собираетесь, телефон тут же пришлет локальное push-уведомление: *«Внимание! Ожидаются дожди в месте вашей поездки. Не забудьте взять зонт!»*.
  
### Что в итоге получится?
На выходе мы получим быстрое, легкое и визуально приятное приложение с современным интерфейсом (на **Jetpack Compose**), которое автоматически страхует пользователя от капризов погоды в любых поездках.
